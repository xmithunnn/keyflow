What's new in Keyflow - latest first.

v2.2 May 2025 Major
New
+Cursor drift mode - occasional backtrack-and-retype sequences for maximum realism
+Session chunks - simulate multiple save sessions with configurable inter-session pauses
+Server-side license validation via Supabase (replaces local key hashing)
+Device fingerprinting for per-device license enforcement
+24-hour license cache so the extension works offline after first validation
Improved
↑Redesigned popup UI - cleaner layout, iOS-style toggles, new hero with comet animation
↑Mistake simulation now distinguishes adjacent-key errors from random transpositions
↑WPM range expanded to 15–300
↑Progress bar now shows real-time ETA
Fixed
✕Google Docs: characters no longer double-fire on certain keyboard layouts
✕Typing no longer silently fails when the active tab changes mid-run
v2.1 March 2025 Minor
New
+Paragraph pause toggle - configurable longer break between paragraphs
+Speed presets (Careful / Normal / Fast / Rush)
+Estimated time display before run starts
Improved
↑Human rhythm engine now adds micro-jitter to comma pauses
↑Start delay extended to 10 seconds option
Fixed
✕Notion: contenteditable blocks now receive keystrokes correctly
✕Stop button no longer leaves a hanging keystroke on fast WPM
v2.0 January 2025 Major
New
+Full rewrite in Manifest V3
+Paid tier system with Stripe checkout
+License key activation inside extension popup
+Free tier: 2 runs per week, no account needed
+Platform detection chip (Google Docs, Canvas, Turnitin, etc.)
Improved
↑Rebuilt typing engine - variable inter-keystroke timing instead of fixed intervals
↑Mistake correction is now more natural (pause → backspace → retype rather than instant swap)
