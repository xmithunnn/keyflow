Setup

Install Keyflow in Chrome

Takes under a minute. Follow the steps below and run the install script so the extension stays working even after you clean up your downloads.

Download keyflow-extension.zip
1

Download and unzip

Click the button above. Find keyflow-extension.zip in your Downloads folder and unzip it. You will see a folder with an extension subfolder plus two install scripts.

2

Run the install script (important)

This step copies the extension to a permanent location so Chrome does not lose it when you delete the zip.

Mac: Open Terminal, drag INSTALL-MAC.sh into it, press Enter.
Windows: Double-click INSTALL-WINDOWS.bat and click Run anyway if prompted.

If you skip this step and load the extension directly from your Downloads folder, deleting or moving the zip will break the extension in Chrome. Always run the script first.

The script copies the extension to:
Mac: ~/Library/Application Support/Keyflow/extension
Windows: %APPDATA%\Keyflow\extension

3

Open Chrome extensions

In Chrome, go to chrome://extensions in the address bar, or open the menu (three dots) and choose Extensions.

4

Enable Developer Mode

In the top right corner of the extensions page, toggle on Developer mode. Three new buttons will appear at the top left.

5

Load the extension

Click Load unpacked. Navigate to the permanent folder the install script created (not your Downloads folder). Select the extension folder inside it. Click Select.

You should see the Keyflow card appear in your extensions list. Pin it to your toolbar by clicking the puzzle icon and pinning Keyflow.
6

You are done

Click the Keyflow icon in your toolbar. The popup opens. Paste text, set your WPM, click Start Typing. If you have a paid license, tap Upgrade to Pro and enter your key to unlock unlimited runs.

Troubleshooting
The extension disappeared after I deleted the zip.
You loaded it from your Downloads folder. Re-download, run the install script, then reload unpacked from the new permanent path.
Chrome says the extension is corrupted or disabled.
Re-download the latest zip from keyflow.live, run the install script, and load unpacked again. Remove the old entry from chrome://extensions first.
Keyflow is not typing in Google Docs.
Click directly into a paragraph in the doc before pressing Start. Make sure the Keyflow popup is still open when you click into the doc. Do not close the popup before the run starts.
I got an error on INSTALL-MAC.sh: permission denied.
Run chmod +x INSTALL-MAC.sh in Terminal first, then run the script again.

Email hello@keyflow.live and describe what step you are on. We usually reply within a few hours.
