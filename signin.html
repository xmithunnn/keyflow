Install guide

Step-by-step setup for the Chrome extension.

1. Download
Click "Download Free" on the homepage
2. Unzip & move
Extract the zip. Run the INSTALL script to save to a permanent folder - do not load directly from Downloads
3. Load in Chrome
Go to chrome://extensions → Enable Developer Mode → Load Unpacked → select the extension folder
4. Activate
Click the Keyflow icon in Chrome, tap "Upgrade to Pro", enter your key
